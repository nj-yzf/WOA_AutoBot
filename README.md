WOA AutoBot 是一款专为《World of Airports》世界机场游戏开发的自动化辅助工具。旨在通过图像识别技术，帮助玩家处理繁琐的机场地面任务，提升挂机效率。本项目大量使用了AI工具辅助开发。

---

## 🌟 核心特性

- **极致速度**：支持 `nemu_ipc` (MuMu 专用) 和 `DroidCast_raw` 高速截图技术，识别响应极快。
- **多种触控**：支持 `minitouch` 毫秒级触控，远超标准 ADB 模拟点击。
- **拟人化设计**：内置“随机思考时间”、“随机任务序列”和“随机滑动轨迹”，大幅降低封号风险。
- **全自动化**：自动领取免费地勤、自动购买车辆、自动延时塔台。
- **多端兼容**：适配 MuMu12/6、雷电、蓝叠等主流 Windows 安卓模拟器。

---

## 🛠️ 环境准备

1. **操作系统**：Windows 10/11
2. **模拟器**：强烈推荐使用 **MuMu 模拟器 12**。
3. **分辨率设置**：模拟器必须设置为 **1600x900 (横屏)**。
4. **游戏语言**：必须设置为 **简体中文**。

---

## 🚀 快速开始

### 方式一：直接运行 (推荐)
1. 从 [Releases](../../releases) 页面下载打包好的 `WOA_AutoBot.exe`。
2. 运行模拟器并进入游戏机场界面。
3. 打开脚本，点击“智能扫描”，选择你的模拟器。
4. 在游戏中打开“待处理飞机”列表（建议仅筛选带黄色感叹号的任务）。
5. 点击“启动脚本”。
